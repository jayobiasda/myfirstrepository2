HTML Elements — Three Divisions

Grouped into Text, Media, and Form elements for quick reference

Text Elements

Structure and typographic building blocks

  • Headings (<h1>–<h6>)
    Document titles and subsections
    semantics
  • Paragraph (<p>)
    Standard text block
    content
  • Blockquote (<blockquote>)
    Quoted sections from other sources
    citation
  • Inline code (<code>, <kbd>)
    Mark shorter code or keyboard input
    inline

Media Elements

Images, audio, video and embedded content

  • Image (<img>)
    Add pictures with alt text for accessibility
    visual
  • Video (<video>)
    Native playback with controls and sources
    interactive
  • Audio (<audio>)
    Embed sound clips and music; support fallbacks
    audio
  • Embeds (<iframe>, <embed>)
    Third‑party widgets, maps, or media players
    external

Form Elements

Inputs and controls for user interaction

  • Input (<input>)
    Text, email, password, number and more types
    controls
  • Select (<select>)
    Dropdown choices and option groups
    choices
  • Buttons (<button>)
    Actions, submit, reset, and custom handlers
    actions
  • Textarea (<textarea>)
    Multi‑line input for long text
    input
